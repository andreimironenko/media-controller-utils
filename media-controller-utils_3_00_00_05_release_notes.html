Introduction

Media Controller Utils are a set of utility programs and libraries that provide a SoC wide set of functions.

The Media Controller Utils product provides the following services to frameworks and applications:

  • Firmware Loader
  • Sys Top
  • PRCM Configuration
  • Loader Memory Configuration

SYS/BIOS operating system is expected to be running on the slaves with all of these platforms.

Version Information

This product's version follows a version format, M.mm.pp.bb, where M is a single digit Major number, mm is 2 digit minor number, pp is a 2 digit patch number, and b is an unrestricted set of digits used as an incrementing build counter.

To support multiple side-by-side installations of the product, the product version is encoded in the top level directory, ex. media_controller_utils_2_03_00_12.

Subsequent releases of patch upgrades will be identified by the patch number, ex. Media Controller Utils 2.02.01.XX with directory media_controller_utils_2_03_01_XX. Typically, these patches only include critical bug fixes.
